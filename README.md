# Olden Era Template Editor

## English

A visual editor for Olden Era (Heroes of Might and Magic: Olden Era) random map generation (RMG) templates.

### Features

- **Visual Topology Editor** — Drag-and-drop zone nodes and connections on canvas
- **Property Panel** — Edit zone/connection/template properties in side panel
- **Direct JSON I/O** — Open and save `.rmg.json` files compatible with the game
- **Global Settings Tab** — Edit game rules, win conditions, value overrides, etc.

## 中文

为英雄无敌：上古纪元 (Olden Era) 随机地图生成 (RMG) 模板打造的可视化编辑器。

### 功能

- **可视化拓扑编辑** — 画布上拖拽区域节点与连线
- **属性面板** — 侧边栏编辑区域/连接/模板属性
- **直接 JSON 读写** — 打开和保存与游戏兼容的 `.rmg.json` 文件
- **全局设置页** — 编辑游戏规则、胜利条件、守卫值覆盖等
